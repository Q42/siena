h1. Write your first Siena model

You've never written a model, here is the simplest model:

h2. <a name="code">The Code first</a>

h3. <a name="model">The Model</a>

bc.. public class Person extends Model {

	@Id
	public Long id;
	
	public String firstName;
	
	public String lastName;
}
	
p. 

h3. <a name="model">Create an entity & Persist it</a>

bc.. Person bob = new Person();
bob.firstName = "Bob";
bob.lastName = "Smith"
bob.insert(); 

p. 

h3. <a name="usage1">Retrieve your entity using the get() function & update it</a>

bc.. Person bob2 = new Person();
bob2.id = bob.id;
bob2.get();
bob2.firstName = "bob2";
bob2.update();

p. 

h3. <a name="usage2">Retrieve your entity by the name & update it</a>

bc.. Person bob2 = Model.all(Person.class).filter("firstName", "Bob").get();
bob2.firstName = "bob2";
bob2.update();

p. 

h3. <a name="usage3">Retrieve your entity by the id directly & delete it</a>

bc.. Person bob3 = Model.getByKey(Person.class, bob.id);
bob3.delete();

p.
